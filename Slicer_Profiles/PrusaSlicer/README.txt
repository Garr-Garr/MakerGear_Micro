How to import the MakerGear Micro profiles within PrusaSlicer:

Step 1: Install PrusaSlicer 

	Go to the following web page to download the program
		
		https://www.prusa3d.com/page/prusaslicer_424/


Step 2: Navigate to the following folder on your computer:

	Windows C: -> Users -> username -> AppData -> Roaming -> PrusaSlicer -> vendor
	Mac “HDD” ▸ Users -> username -> Library -> Application Support -> PrusaSlicer -> vendor
	Linux ~/.PrusaSlicer/vendor

Step 3: Copy the contents of the SD Card folder "vendor" to your computer's local PrusaSlicer "vendor" folder

Step 4: Launch PrusaSlicer

Step 5: Open the "Configuration" menu and select "Configuration Wizard"

Step 6: Select the "Other Vendors" menu and enable "MakerGear"

	A new configuration wizard tab should appear

Step 7: Open the new "MakerGear FFF" tab and add the "MakerGear Micro"
